# **App Name**: Ritzee Wear

## Core Features:

- 3D Product Viewer: Display interactive 3D models of t-shirts using Three.js. Users can rotate, zoom, and interact with the models in 360 degrees. GLB files are loaded dynamically based on the product.
- Dynamic Hero Section: Enable non-technical admins to update the hero section content (text, images, layout config) through a custom Page Builder backend without needing code changes.
- Odoo Integration: Connect with Odoo API for real-time inventory management and to store GLB file references. Frontend fetches GLB file URL from Odoo.
- Shopify Integration: Use Shopify Storefront API for managing the cart, checkout process, and customer authentication, ensuring a seamless shopping experience.
- Inventory Sync Tool: Tool which uses inventory data from Odoo and automatically ensures stock levels and availability on the Shopify storefront are in sync to avoid overselling and ensure customer satisfaction.

## Style Guidelines:

- Primary color: Acid green (#BEF264) to evoke a sense of Y2K futurism and high-energy fashion.
- Background color: Deep charcoal (#1E1E1E) to provide a high-contrast backdrop for the acid green accents.
- Accent color: Electric purple (#A21CAF) to complement the acid green and add a futuristic, eye-catching element.
- Headline font: 'Space Grotesk', a sans-serif font for headlines, providing a modern, tech-inspired feel.
- Body font: 'Inter', a grotesque sans-serif font for body text, providing a clean and readable appearance.
- Incorporate pixelated icons and graphics reminiscent of Y2K aesthetics.
- Implement a neo-brutalism design with bold typography, overlapping elements, and asymmetrical layouts for an edgy, Gen Z aesthetic.
- Integrate fluid animations and micro-interactions, utilizing Lenis scroll for a smooth, dynamic user experience.
